# Hardware Requirements

Dieses Projekt ist vollständig lokal ausführbar und benötigt keine spezielle Hardware.

## Mindestanforderungen

- CPU: Jede moderne x86_64 oder ARM CPU
- RAM: 4 GB
- Speicherplatz: < 200 MB
- Betriebssystem: Windows, macOS oder Linux
- Python: Version 3.10 oder höher

## Keine GPU erforderlich

Dieses Projekt benötigt keine GPU und verwendet keine CUDA-, ROCm- oder TPU-Beschleunigung.

## Keine externen Dienste

- keine Cloud-APIs
- keine externen LLMs
- keine Internetverbindung erforderlich

## Abhängigkeiten

Dieses Projekt verwendet ausschließlich:
- Python-Standardbibliothek

## Ziel

Alle Beispiele sollen auf normaler Consumer-Hardware lauffähig sein.
